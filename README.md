

### 前言

在很久很久以前,大概大一寒假的时候我就想做一个自己的网站,但是由于多方面因素的影响,一直未能成行。现在趁着暑假空暇,终于做了一个关于校园的二手书交易平台。

本项目是一个完全的服务端项目,各模块分类明确,注释清晰,初学者完全可以用来练习巩固知识,也可用做毕设项目。
